This level gives you a glimpse of what you might do in the future. For now though, just use your self.move() methods. HINT Use your numbers inside parens to repeat methods.

Bonus

Functions are like mini code blocks, that can be reused. Here is an example: 

def afunction():
  self.moveUp()
  self.moveRight()

This is a function being called on: afunction()
Calling on a function runs all the code within it.

Now you try!

First, see what code you have that can be repeated. In this case, you need to move down then right. Define (def in code) a function that does this. NOTE: functionsALWAYS come first in your code, otherwise, it is run before it knows what it is!

CHALLENGE: Using your knowledge of functions and repeating commands, complete the level using a function and 5 other lines of code!
