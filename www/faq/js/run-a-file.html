How to run a JavaScript program


FAQ >> JavaScript >> Run a program

Once you write your glorious bit of JavaScript magic, you probably want to run it. You do this by executing the following commands in the main command line.

chmod +x file This makes it so the computer knows to run it. file or ./file This will actually run it.

Bonus


node file.js

The difference is one file ends with a .js, and this is the only way to run a file with .js. You can still use this if it doesn't end with .js, but why would you want to type that all the time?
