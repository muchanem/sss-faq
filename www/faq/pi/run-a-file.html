How to run a Shell/Bash program


FAQ >> Pi >> Run a program

Once you write your glorious bit of shell magic, you probably want to run it. You do this by executing the following commands in the main command line.

chmod u+x file This makes it so the computer knows to run it. file or ./file This will actually run it.
